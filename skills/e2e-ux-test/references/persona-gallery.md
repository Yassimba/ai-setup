# Persona gallery & card template

## Card template

Save as `ai-docs/ux-personas/<slug>.md`. The runs ledger is what makes reruns regression passes.

```markdown
# <Name> — <product> (<surface>)

1. Hardest user: <who, age, role, how they do it "the old way">
2. Tech comfort: <the lower the better — which tools they actually use. Dev surfaces invert this: high skill, zero patience>
3. The ONE task: <their core job, one sentence>
4. Gives up when: <effort, jargon, slowness, confusion — their breaking point, in the surface's effort unit>
5. Frustrated voice: <blunt / sweary / dismissive / sighing — with a sample line>

## Runs
| date | effort to core task | verdict | RED count |
|------|--------------------:|---------|----------:|
```

## Worked example

> **"Big Mick" McAllister** — 58-year-old S&C coach. Uses WhatsApp and that's it. His "spreadsheet" is a paper notebook. "If I can't figure it out in 10 seconds I'm going back to my notebook." Needs to log session results for 25 players. Hates small text, jargon, and passwords.
